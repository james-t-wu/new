1. UserWay 网络无障碍人工智能技术公司

2023年12月31日，据 Calcalist 报道，Level Access 以 9900 万美元收购 Accessibility AI 技术开发商 UserWay，两家公司已签署最终协议。收购以全现金交易，UserWay 股东将获得每股21.06美元，相比市场价格有显著溢价。该交易预计将于 2024 年初完成。

UserWay 成立于2016年，并于2022年1月在特拉维夫证券交易所上市。UserWay 是一家数字无障碍软件解决方案全方位服务供应商，用以提高网站对于残疾人士的可访问性和可用性，其服务被全球数以百万计的网站使用。该公司的人工智能技术帮助网站、应用程序和数字文档更轻松地符合无障碍法规。

Level Access 总部位于阿灵顿，专门进行网页无障碍测试，专注于提供数字无障碍解决方案。

本次收购旨在结合 UserWay 的 AI 技术和 Level Access 的专业知识，使数字内容更加无障碍。收购后，公司将成为 Level Access 旗下的一家私有企业，继续以 UserWay 品牌运营。

2. Atom Security 网络安全公司

2023年12月24日，据 Calcalist 报道，Mend.io 收购了以色列网络安全公司 Atom Security。收购金额约数百万美元。

Atom Security 成立于2023年初，专注于网络风险评估。

以色列公司 Mend.io（前身为 WhiteSource）成立于2011年，致力于帮助开发者管理开源库并减轻安全风险，迄今已筹集了 1.22 亿美元资金，在过去三年中进行了五次收购。

赛门铁克 2023 年的一份报告显示，74% 的企业攻击针对应用层。将 Atom Security 的技术集成到 Mend.io 的产品线中预计将扩大安全覆盖范围并将不相关发现的数量减少 60% 至 70%。本次收购完成后，Atom Security 将加入 Mend 的云原生技术部门。

注：网络安全中的“不相关发现（irrelevant findings）”通常是指在安全评估或自动扫描过程中发现的误报或非关键问题。安全工具和程序扫描系统、网络或应用程序是否存在漏洞和潜在威胁时，并非所有发现都同样重要。误报是指这些发现被标记为潜在的安全威胁，但经过进一步调查，发现是良性的，例如，合法软件可能会表现出与恶意软件类似的行为，导致安全工具错误地标记它。非关键问题是指这些是真实的发现，但它们可能不会对组织构成重大威胁，例如，不太可能被利用或影响最小的次要软件漏洞。减少不相关发现对于网络安全效率至关重要，大量不相关发现可能会导致资源和时间的浪费，因为安全团队必须调查每个警报。此外，过多的误报可能会导致警报疲劳，由于大量的非关键警报，重要的威胁可能会被忽视。

3. Spera Security 身份安全平台

2023年12月19日，Okta 宣布收购 Spera Security，一家总部位于以色列特拉维夫的专注于身份安全的初创公司。收购最终金额尚未披露，据估大约1亿到1.3亿美元，预计交易将于2024年2月初完成。

Spera 专门提供针对基于身份攻击的主动、持续保护，包括被盗用的凭证、社交工程和网络钓鱼。平台提供用于识别和管理云和本地环境中风险的工具，收购前已筹集1000万美元。

Okta 是美国知名的身份和访问管理公司，服务包括威胁检测和安全姿态管理。收购 Spera Security 是 Okta 加强其身份威胁检测和响应（ITDR, Identity Threat Detection and Response）能力战略的一部分，也是这家身份安全巨头公司在以色列的首次收购。

据 SecurityBrief 报道，Okta 认为身份泄露是当今企业面临的普遍威胁媒介，其中 86% 的泄露主要是由网络钓鱼攻击和其他凭据滥用造成的。这包括社会工程（Social Engineering）攻击、撞库和滥用。根据 Gartner 的研究，身份威胁检测和响应（ITDR）的实施是身份安全的关键。据预测，到 2026 年，嵌入式 ITDR 策略将被 90% 的企业采用，比目前的 5%-20% 有显著飞跃。Okta 收购 Spera Security 继续强化其对安全身份产品的承诺，也符合其让每个人都能以安全方式使用任何技术的使命。

注：社会工程攻击是一种利用人为错误来获取私人信息、访问权限或贵重物品的操纵技术。与通常依赖技术缺陷的传统黑客攻击不同，社会工程攻击针对安全系统的人为因素。

4. DGN Games 社交及移动游戏公司

2023年12月14日，据印媒 Inc42 报道，印度游戏公司 Octro 宣布收购 DGN Games，收购金额尚未公开。

DGN Games 是一家总部位于以色列的社交和移动游戏公司。该公司以其沉浸式老虎机体验而闻名，如 Lucky Time Slots 和 Old Vegas Slots。

Octro 成立于2006年，由 Saurabh Aggarwal 创立，是一家游戏开发工作室，主要为用户提供在线娱乐，游戏覆盖超过2亿玩家。产品组合包括 Teen Patti、Play Rummy 和 Octro Poker 等热门游戏。

此次收购使 Octro 能够扩展其在关键国际市场如澳大利亚和美国的影响力，并开拓新的人群，以推动增长。

Octro 首席财务官 Deepak Goyal 表示，此次收购将帮助公司实现显著的成本效率。此外，双方公司之间的协同作用将使 Octro 能够获得 DGN 的技术专长和人才库。收购还将促进两家公司的综合实力，开发面向全球观众的游戏，提升玩家的游戏体验。

2023年12月5日，据 techawkng 报道，Appsflyer 宣布收购以色列初创公司 Oolo，收购金额尚未披露。

Oolo 是一个专注于用户获取和变现决策的 AI 驱动平台。公司提供创新的广告监测解决方案，以增强增长策略，提供可操作的洞察和警报，以优化广告支出回报（ROAS）、媒体收入和留存率。

以色列独角兽企业 Appsflyer 以其市场测量、归因和数据分析而知名。

本次收购旨在转变业务的 AI 驱动决策制定，特别是在营销、增长和货币化策略方面。Oolo 将作为 oolo by AppsFlyer 运营，整合进 Appsflyer 的隐私云市场，这一战略举措符合全球范围内，包括非洲在内的多个市场中 AI 技术的不断整合。Oolo 和 Devtodev (Appsflyer 上月收购了立陶宛游戏和分析公司Devtodev) 将成为 AppsFlyer 最近推出的隐私云的首批应用程序之一。

6. Raftt 云平台服务

2023年12月4日，据 Calcalist 报道，Wiz 以数千万美元收购了以色列云平台服务公司 Raftt。

Raftt 成立于2020年，专注于创建和共享开发环境的云平台，此前已获得500万美元投资。Raftt 专注于开发过程，提供云端运行的开发人员环境，支持在使用本地 IDE 和其他工具及工作流程的同时进行协作。开发人员不但可以继续使用他们喜欢的工具和环境，同时还受益于 Raftt 提供的基于云的协作和环境管理，这有助于保持开发过程的一致性和效率。

Wiz 是一家专注于云安全的独角兽公司，成立于大约四年前，目前估值已达到100亿美元，年收入超过2.5亿美元，这是其成立以来的首次收购。

此次收购旨在结合 Raftt 的开发平台和 Wiz 的云安全专长，解决软件开发中的安全挑战。Raftt 的团队将加入 Wiz，预计不久将有新产品整合推出。

IDE（集成开发环境）是指为计算机程序员进行软件开发提供综合设施的软件应用程序。通常包括源代码编辑器、构建自动化工具和调试器。一些 IDE 还包含用于版本控制和开发图形用户界面 (GUI) 的工具。IDE 通过将各种开发工具集成到单个应用程序中，使编写和测试软件的过程更加高效。这可以包括智能代码完成、错误诊断和更正等功能，以及从 IDE 本身编译和运行应用程序的能力。
