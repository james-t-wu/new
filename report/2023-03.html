以色列收购事件

1. Lightspin - 云安全公司

2023 年 3 月 29 日，美国数字通信巨头思科（NASDAQ: CSCO）宣布将收购以色列初创公司 Lightspin。收购金额尚未披露，市场信息认为交易金额约 2 亿美元。

在过去 20 年里，思科在以色列进行了多起网络交换机和硬件方面的收购，近年来的收购范围扩大到网络安全和云安全，并扩大了其在特拉维夫的办事处。Lightspin 提供了一个整体解决方案，可跨云原生资源提供端到端的云安全态势管理 (CSPM)。思科表示双方的共同目标是通过从构建到运行时的端到端安全性和可观察性帮助客户实现云环境现代化。

2. Aspecto - OpenTelemetry 提供商

2023 年 3 月 21 日，美国软件开发和可视化工具提供商 SmartBear 宣布收购 OpenTelemetry (OTel）先行者 Aspecto。收购金额尚未披露。

Aspecto 成立于 2019 年，为开发人员提供分布式跟踪体验，自成立以来一直在推动和定义OTel项目和社区方面发挥重要作用。其可观察平台提供了对分布式服务中发生的每个交互和性能问题的可见性，使开发人员能够解决瓶颈和错误，同时跨跟踪、日志和指标关联根本原因。

通过此次收购，SmartBear 将为开发团队提供跨软件开发生命周期 (SDLC) 的端到端可见性。开发人员可以使用 Aspecto的基于 OpenTelemetry 的可观察性平台来可视化复杂的跟踪，优化性能，提供与 API 的更好协作，更好地探索微服务。这种可见性将使开发人员能够构建具有竞争力的应用程序，并有信心更快、更频繁地进行部署。

3. Minerva Labs - 反逃避和勒索软件预防技术提供商

2023年 3 月 15 日，总部位于波士顿的云风险和威胁检测领域的领导者 Rapid7（NASDAQ：RPD）宣布已以 3800 万美元的现金加股票交易收购了反逃避和勒索软件预防技术领先提供商以色列公司 Minerva Labs，此次收购旨在提升其托管检测和响应 (MDR) 平台的附加功能，以阻止攻击者规避策略和防止恶意软件部署。

Minerva Labs 成立于 2014 年，帮助用户降低与勒索软件相关的风险。公司拥有 15 名员工，主要由工程师组成。

4. Even-Ari Green - 栽培生长介质制造商

2023年3月13日，Pelemix 宣布已收购总部位于以色列的栽培生长介质制造商 Even-Ari Green Ltd.（EAG）。收购金额尚未披露。

Pelemix 是水培种植和苗圃用椰糠-椰糠泥炭介质市场的领导者，其椰糠介质在全球范围内用作水培作物、植物繁殖、水果和蔬菜生产、大麻种植、盆栽植物和花坛植物生产的生长介质，主要面向专业种植者市场。

5. Axis Security - 云安全提供商

2023 年 3 月 2 日，HPE（NASDAQ: HPE）宣布已就收购云安全提供商 Axis Security 达成最终协议。交易金额尚未披露，有市场信息透露总额约为5亿美元，HPE 未对此信息进行肯定。

HPE（Hewlett Packard Enterprise）从惠普公司拆分，此次收购将使HPE能够通过提供统一的安全访问服务边缘（SASE）解决方案来扩展其边缘到云的安全功能，以满足对作为服务交付的集成网络和安全解决方案日益增长的需求。收购后，Axis Security 将成为 HPE Aruba Networking 部门的一部分。

Axis Security 位于以色列特拉维夫。公司提供了一个云原生SSE平台Atmos，该平台提供经过身份验证的用户对网络边缘私有应用程序的访问，一个安全Web网关（SWG）以保护用户对互联网的访问，一个云访问安全代理（CASB）提供对SaaS应用程序的安全在线访问，以及数字体验监控（DEM）以深入了解用户体验。
