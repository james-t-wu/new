报告概况

2023年10月，以色列发生 28 个投资收购项目，已公开总额约为 7.69 亿美元；其中投资项目 25 个，已公开投资总额约 7.5 亿美元。被收购以色列公司包括：癌症治疗生物技术公司 VBL Therapeutics 、无线解决方案提供商 Siklu、Dig Security 云数据安全公司。

以色列收购事件

1. Dig Security - 云数据安全公司

2023年10月31日，全球领先的网络安全公司 Palo Alto Networks 宣布收购数据安全姿态管理技术公司 Dig Security。收购金额尚未披露。

Palo Alto Networks 提供下一代网络安全服务，以威胁情报和自动化技术支持其最佳网络安全平台和服务。

Dig Security 是一家创新的云数据安全公司，2021年在以色列成立。

此次收购 Palo Alto Networks 计划将 Dig Security 的 DSPM 技术整合到 Palo 的Prisma Cloud 平台中，以增强 Palo 在云数据安全领域的能力，帮助企业发现、分类、监控和保护所有云数据存储中的敏感数据，从而为用户提供对多云数据资产的可见性和控制。

2. Siklu - 无线解决方案提供商

2023年10月24日，Ceragon Networks Ltd. 宣布已签订最终协议以 1300 万至 1500 万美元的企业价值收购 Siklu。

Ceragon Networks 是全球 5G 无线传输解决方案提供商，帮助全球运营商和其他服务商提高运营效率，并通过创新的无线回传和前传解决方案提升最终用户的体验质量。

Siklu的无线解决方案一直在毫米波频段，提供多千兆位“无线光纤”连接，被服务商和系统集成商用来提供 5G 千兆无线接入服务。

3. VBL Therapeutics - 癌症治疗生物技术公司

2023年10月16日，Notable Labs 宣布已与 VBL Therapeutics 完成合并。

Notable Labs 是一家临床阶段治疗平台公司，利用专有平台 PPMP（通过生物模拟和预测分析来确定癌症患者对特定治疗的反应，加速临床开发过程）为癌症患者开发预测精准药物。

VBL Therapeutics (NASDAQ: VBLT) 是以色列一家生物制药公司，专注于开发免疫炎症性疾病的靶向疗法。
