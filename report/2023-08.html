报告概况

以色列收购事件

1. iNNOGING Medical - 远程离线查看和分析超声扫描工具

2023年8月28日，医疗模拟产品领先制造商 3B Scientific 宣布已收购云超声模拟领域公司 iNNOGING Medical。收购金额尚未披露。

iNNOGING Medical 开发了名为 e Sono 的基于云的开创性 SaaS 解决方案。该解决方案有利于超声模拟，允许用户使用任何联网设备从任何位置训练和分析超声扫描。公司成立的愿景是改变超声图像分析培训，力求增强全球超声诊断能力。

3B Scientific是医疗模拟器和教育领域的全球领导者，致力于推动全球医疗和保健教育的发展。此次收购旨在增强 3B Scientific 在医学模拟和培训领域的产品和服务。

此次收购标志着两个扎根于医疗培训和模拟领域的实体的合并。凭借 iNNOGING Medical 先进的超声模拟技术和 3B Scientific 在医学教育领域的广泛影响力和专业知识，这一合作伙伴关系有望为医疗保健教育和培训带来革命性的变化。

2. Perimeter 81 - 网络即服务

2023年8月10日，网络安全公司 Check Point Software Technologies (NASDAQ: CHKP) 已签署最终协议，收购创新型安全服务边缘 (SSE) 公司 Perimeter 81。收购预计于2023年第三季度完成，收购价值约 4.9 亿美元。

Perimeter 81成立于2018年，提供强大且易于使用的融合网络和安全平台，将办公室或远程的所有用户连接到位于本地或云端的所有企业资源。公司还支持一小时部署，解决了传统方案需要数周时间的问题，强调易用性和简化管理。

Check Point 起源于以色列，总部位于美国加利福尼亚州，专注于企业网络安全的各个方面，从云服务到人工智能，和主动威胁检测支持的网络安全。

混合办公（Hybrid work）已逐渐成为全球企业的标配，数字化转型正在不断将数据中心转移到云端，这一转变需要一种先进的网络安全方法，以无缝适应和响应现代分布式企业的需求。Check Point 的此次收购将增强其针对远程和混合办公人员的工具。

3. Laminar - 数据安全态势管理平台

2023年8月8日，数据安全供应商 Rubrik 宣布签署协议收购数据安全态势管理 (DSPM, Data Security Posture Management) 平台 Laminar。收购金额尚未披露。

Laminar Technologies 于 2020 年在特拉维夫成立，提供专为公共云环境和基于云的数据仓库的数据安全平台。平台拥有关键的数据安全功能，例如状态管理、访问治理、威胁监控和威胁响应。在收购之前，Laminar 已成功筹集了总计 6700 万美元的资金。

Rubrik Inc. 是一家领先的数据安全供应商，致力于其数据保护软件的零信任架构，强调严格的用户访问要求和增强的组织控制。此次收购旨在增强 Rubrik 的数据安全产品，使其能够提供全面的网络态势和网络恢复解决方案。

企业当今面临的最大挑战之一是日益分散的环境中数据的可见性较低。此次合并将跨企业、云和 SaaS，整合网络恢复和网络态势，打造业界首个完整的网络弹性产品。

4. Webbing - 移动虚拟网络运营商连接提供商

2023年8月7日，全球领先的物联网连接提供商 Wireless Logic 收购了专注于企业移动和物联网应用连接的全球移动虚拟网络运营商 Webbing。收购金额尚未披露，根据市场信息，收购金额超过 2 亿美元。

Webbing 成立于 2010 年，是一家全球移动虚拟网络运营商（MVNO, Mobile Virtual Network Operator)，为企业移动、汽车和物流等行业的全球客户提供领先的连接服务，是 eSIM 技术的市场领导者，引领物联网向新的 GSMA eSIM 标准转变。Webbing 的解决方案（例如 WebbingCTRL）可简化物联网连接、降低成本并缩短上市时间，使企业能够保持对其连接部署的完全控制。

Wireless Logic 是全球著名的物联网连接提供商，近期一直在大举收购，包括 IoThink Solutions、Mobius Networks、Jola 和 Blue Wireless。此次收购 Webbing 的动机是两家公司产品的战略互补性，旨在扩大 Wireless Logic 全球影响力并增强其 eSIM 功能。

5. Spott - 人工智能数据平台

2023年8月3日，金融科技公司Payoneer (NASDAQ: PAYO)，宣布收购以色列实时数据平台 Spott。收购金额尚未披露。

Spott 是一个实时数据平台，提供智能金融产品，保护电子商务在线商家免受意外损失，快速增长、管理风险并实现财务稳定。Spott 利用人工智能有效识别、评估和量化数据，确保企业做出明智迅速的决策。

Payoneer 致力于帮助中小企业（尤其是新兴市场的中小型企业）在全球范围内进行交易、管理资金、支付和发展业务。Payoneer 将利用 Spott 简化决策过程，该技术的首次应用将在 Payoneer 营运资金产品上，以增强承保能力。

6. Castory - 无缝建筑施工进度跟踪和文档平台

2023年8月1日， 建筑项目管理系统公司 Top Ramdor 宣布收购 Castory-ai。Castory-ai 以其针对建筑项目的创新 3D 成像解决方案而闻名。公司技术利用头盔摄像头和无人机，提供对施工现场的实时洞察，确保项目管理能够监控进度并及早发现潜在问题；其人工智能功能可以自动识别和报告施工进度，缩小现场活动和办公室管理之间的差距。

Top Ramdor提供全面的基于云的解决方案。其解决方案嵌入到众多组织中，伴随着建筑项目的整个生命周期。

此次收购是 Top Ramdor 在以色列众多项目中成功实施 Castory-ai 的解决方案之后进行的。
