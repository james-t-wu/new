以色列收购事件

1. Codefresh - 软件公司

2024 年 2 月 27 日，据 Ctech 报道，以色列软件公司 Codefresh 已被澳大利亚公司 Octopus Deploy 收购。收购金额尚未披露，据估在4000-5000万美元之间。

Codefresh 于 2014 年在以色列成立，它是 Argo 的维护者之一，并在 Kubernetes 的持续交付（CD）、GitOps 和持续集成（CI）领域处于领导地位。Codefresh 的平台专注于提供针对 Kubernetes 应用的自动化构建、测试和部署解决方案，使得软件开发和部署过程更加高效和可靠。

Octopus Deploy 是澳大利亚一家持续部署自动化平台提供商，专注于持续交付（CD）的行业标准平台。

此次战略收购使 Octopus Deploy 成为 DevOps 生态系统中最大的私营公司之一，将 CD、CI 和 GitOps 整合到一个可靠的平台中，进一步推动软件交付的创新和效率。通过这次收购，Octopus Deploy 希望扩大其在美国市场的影响力，并增强其在持续部署工具领域的专业知识。

2. Manpower Language Solutions - 语言和内容解决方案提供商

2024 年 2 月 21 日，据美通社报道，全球本地化提供商和多市场推动者 BLEND 宣布已收购以色列领先的本地化服务提供商 Manpower Language Solutions (MLS)。收购金额尚未披露。

MLS 的专家团队提供广泛的服务——从笔译、口译和本地化到内容写作和编辑。公司为全球各个领域的数千名客户提供服务，包括技术行业（IT、新兴技术、人工智能、网络安全）、金融机构、政府部门、执法机构、流媒体和媒体公司、生命科学、生物技术等。

BLEND 是一家全球性的本地化供应商和市场服务商，提供 120 多种语言服务，作为端到端多市场推动者，帮助全球品牌在全球截然不同的市场中建立本土影响力，通过人工智能驱动技术以及由语言专家和语音人才组成的全球社区的力量，BLEND 可以大规模提供复杂、持续的本地化。

3. Inspekto - 人工智能机器视觉软件公司

2024 年 2 月 15 日，据 metrology 报道，西门子股份公司 (Siemens Aktiengesellschaft) 收购了人工智能机器视觉软件公司 Inspekto。收购金额尚未披露。

Inspekto以创新的人工智能机器视觉技术而闻名，其解决方案旨在为任何规模和类型的制造商在生产线上实现自动化视觉检查。他们的自主机器视觉系统由 AMV-AI 技术提供支持，无需视觉专家或大量生产数据即可部署开箱即用的QA（Quality Assurance，质量保证）系统，增强汽车、电子和消费品等各个行业的质量控制。

此次收购将显着增强西门子在数字工业业务部门工厂自动化领域的能力。

4. Wib Security - 网络安全公司，API 安全平台

2024 年 2 月 11 日，据 Ctech 报道，开发 API 安全平台的网络安全初创公司 Wib 已被美国网络巨头 F5 收购。收购金额尚未披露，据估在数千万美元。

Wib 是一家以色列初创公司，成立于 2021 年，专注于 API 安全。通过在 API 生命周期的每个阶段提供实时检查、管理和控制，Wib 可以实现库存和 API 变更管理的自动化，识别流氓、僵尸和影子 API，并分析业务风险和影响。

F5 是一家美国网络安全公司，提供广泛的解决方案。此次收购的目标是通过整合 Wib 的先进 API 安全技术，加强 F5 分布式云服务，这些技术提供了对 API 及其相关风险的更好可见性、测试和上下文。这是 F5 在以色列的第五次收购，被认为是 API 安全领域的一项重要进展。
