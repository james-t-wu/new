以色列收购事件

1. Speedb - 数据存储引擎

2024 年 3 月 21 日，领先的实时数据平台 Redis 宣布收购以色列数据存储引擎 Speedb。收购金额尚未披露，在被 Redis 收购之前，Speedb 获得了总计 400 万美元融资。

Redis 是一个备受推崇的以色列实时数据平台，专门用于内存数据存储，近期已将其功能扩展到内存数据库功能之外。

Speedb 因与 RocksDB 兼容的高效数据存储引擎而闻名，能显著提高应用程序性能和成本效益，特别是在 DRAM 不是必需的场景中。

Redis 收购 Speedb 旨在增强 Redis 的实时数据处理能力，并扩展其在各种用例（包括生成式 AI 应用程序）中的性能和可扩展性，超越传统的DRAM依赖。Speedb 在 SSD 存储和传输速率方面的先进技术，使Redis 能够满足速度至关重要的企业应用的全方位性能和成本要求。

2. Oxeye Security - 云原生应用程序安全测试解决方案

2024 年 3 月 20 日，软件开放商 GitLab 宣布收购以色列云安全公司 Oxeye。收购具体金额尚未披露，据 Calcalistech 报道约 3000-4000 万美金。这是继 2020 年收购 Fuzzit 后 GitLab 在以色列的第二次收购。

GitLab 被称为最全面的人工智能驱动的 DevSecOps 平台，以其一体化的软件交付方法而著称，已成为超过 3000 万注册用户的信赖平台，包括超过一半的财富 100 强公司。

Oxeye 由 Dean Agron 和 Ron Vider 创立，通过开发先进技术来检测和解决应用程序开发阶段的漏洞而闻名。

此次收购预计将带来多项好处，包括改进的SAST扫描技术，显著减少误报，提供更准确的结果和可操作的见解。这一整合将使开发者和安全专业人员拥有更好的工具进行漏洞管理和修复，进一步简化安全应用程序构建过程。

3. Avalor - 网络安全公司

2024 年 3 月 14 日，据 Ctech 报道，美国网络安全巨头 Zscaler 将以 3.5 亿美元收购以色列初创公司 Avalor。这是 Zscaler 在以色列的第三次收购 （前两次分别为 Canonic 和Trustdome）。

Avalor 2022 年成立，是一家以色列网络安全初创公司，通过其安全数据架构开发了一个专注于风险管理的尖端平台。它擅长通过聚合、规范化和跟踪从发现到补救的风险数据来管理网络安全资产、控制、身份、漏洞等。Avalor 解决方案的独特之处在于它能够集成和处理几乎任何来源、任何格式的数据。自成立以来公司已成功地获得了 3000 万美元的投资，主要投资人来自 Cyberstarts、TCV 等。

Zscaler 是领先的云安全公司，本次战略收购将 Avalor 在数据管理和风险评估方面的专业知识与 Zscaler 强大的云安全解决方案相结合，显著增强 Zscaler 人工智能的能力和实时网络安全洞察。

此次收购预计将活跃网络安全并购市场，体现行业的积极整合，反映了将人工智能和高级数据分析纳入网络安全策略的日益增长的趋势。该领域的其他以色列公司包括 Zenity 等。

4. Flow Security - 云数据运行时安全解决方案

2024 年 3 月 6 日，据 CrowdStrike（NASDAQ：CRWD）公司官网及环球报消息，美国网络安全巨头 CrowdStrike 已同意收购以色列云数据运行时安全解决方案公司 Flow Security。交易细节尚未披露。

Flow Security 被誉为业界第一个也是唯一一个云数据运行时安全解决方案。

CrowdStrike 利用全球最先进的云原生平台重新定义了现代安全性，以保护企业风险的关键领域——端点和云工作负载、身份和数据。

Flow Security 的 DSPM 与 CrowdStrike 的 Falcon 平台的集成预计将重新定义现代数据安全，提供一个统一的平台来保护跨云服务、第三方 API 和 GenAI 应用程序的数据。此次收购解决了保护分散数据的关键挑战，这已成为攻击者利用云生态系统中安全漏洞的主要目标。

Flow Security 之所以能够成为业界第一个也是唯一一个云数据运行时安全解决方案，主要是因为其全面的数据安全方法涵盖了数据处理和环境的各个阶段。其平台旨在发现、分类和保护敏感信息，不仅在静态信息时，而且在公共云、本地基础设施和 SaaS 平台上移动时也如此。
